Validation Dossier

Mapping patient biology to predict treatment response and improve trial outcomes

What We Demonstrate

  • Patient biology organizes into a stable coordinate system
  • Treatment response follows geometric structure
  • Patients can be ranked without dataset-specific training
  • Responder enrichment improves across independent datasets

System Overview

Transcriptomic data (~18,000 genes) is projected into a fixed biological space. All downstream analysis operates within this coordinate system.

Fixed mapping. No retraining.

Biological Coordinate System

Four primary axes:

  • Inflammatory (z1)
  • Neuro-modulatory (z2)
  • Metabolic (z3)
  • Systemic state (z4)

Diseases are trajectories — not separate systems

Response Mechanism

  • Directional (alignment-driven response)
  • Diffuse (global system shift)

Different therapies operate under different regimes

Validation Protocol

  • Strict out-of-fold evaluation
  • No dataset-specific tuning
  • Independent cohort validation

Robustness & Generalization

  • Rank stability (Spearman ≈ 0.78)
  • Cross-modality (array + RNA-seq)
  • Cross-disease consistency

Negative Controls

  • Label shuffle → performance collapses
  • Feature permutation → structure collapses

Confirms biological signal, not statistical artifact

System Boundaries

  • Lower signal in highly heterogeneous diseases
  • Requires sufficient cohort size
  • Acute diseases behave differently
