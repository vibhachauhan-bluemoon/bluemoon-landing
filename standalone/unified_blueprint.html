0. Data Ingestion & Immutable Coordinate Lock

The entire foundational validity of the BlueMoon system relies on bitwise reproducibility during raw data ingestion. Any deviation during the mapping of X_raw to the predefined latent space causes systemic collapse of the generative weights.

The Four Golden Rules of Pre-Projection

  1. Immutable Gene Alignment: The transcriptomic input matrix must be strictly reindexed to match the exact canonical 18,318 gene array. No substitution or fuzzy mapping is permitted.
  2. Zero-Fill Protocol: Missing genetic reads in external cohorts must never throw exceptions or trigger row-drops. They must be handled via a hard reindex + fillna(0) operation.
  3. Gene-Wise Z-Score: Prior to matrix multiplication with W.npy, the raw counts must undergo a strict gene-wise z-score normalization.
  4. Frozen Latent Scale: The post-projection array must be scaled using the fixed latent_scale.npy vector. Never recompute the latent scale per dataset. Doing so destroys the coordinate system and introduces catastrophic data leakage.

4. Vectorial vs. Scalar Response Regimes (Biological Kinematics)

The stratification pipeline assumes different interventions act on the biological space via fundamentally different mathematical mechanisms:

Vectorial / Directional Regimes

These are rapid, mechanism-specific interventions (e.g., Ketamine in MDD, Infliximab in Crohn's). They force patients to align strictly along isolated axes. Mathematical tracking requires specific dimensional distance vectors (score_dir focus).

Scalar / Diffuse Regimes

These are slow-acting, system-wide interventions (e.g., SSRIs). They cause pervasive, relative target reductions rather than absolute positional shifts. The engine maps these via overarching systemic contraction (score_axis mapping).

6. Core Scoring Algorithms

A. Non-Linear Hybrid (Biological Ceiling Target)

Extracts the maximum biological signal limits by isolating Responder density dynamically and interpolating cross-axial alignment.

def score_hybrid(z1n, z2n, z3n, z4n, is_R, lam=0.5, beta=0.6):
    # 1. Optimal directional vector toward responders
    w = np.array([np.mean(z1n[is_R]), np.mean(z2n[is_R]), np.mean(z3n[is_R])])
    w = w / np.linalg.norm(w)
    score_dir = w[0]*z1n + w[1]*z2n + w[2]*z3n
    
    # 2. Maximum absolute axis deviation constraint
    score_axis = np.max(np.vstack([z1n, z2n, z3n]), axis=0)
    
    # 3. Blended projection and systemic sigmoid suppression
    score_idx = lam * score_dir + (1 - lam) * score_axis
    score_decision = score_idx * (1 - beta * sigmoid(z4n))
    return score_decision

B. OOF Linear + Refined (Audit-Safe Deployed Product)

A formalized linear vector tracking model guaranteeing separation cleanly out-of-fold. Prevents label leakage by calculating vectors only on train splits, rejecting heterogeneous "soft" non-responders via geometric clustering.

def score_refined_oof(Z_train, y_train, Z_test):
    mu_R = Z_train[y_train == 1].mean(axis=0)
    NR_train = Z_train[y_train == 0]
    
    # Standard Projection
    w_proj = mu_R - NR_train.mean(axis=0)
    
    # Isolate structurally clustered Non-Responders (NR-Hard)
    dist_NR = np.linalg.norm(NR_train - mu_R, axis=1)
    k_nr = max(1, int(0.3 * len(NR_train)))
    NR_hard = NR_train[np.argsort(dist_NR)[:k_nr]]
    
    # Refined target formulation
    w_ref = mu_R - NR_hard.mean(axis=0)
    
    # Execution across unseen test split
    score_proj = Z_test @ w_proj
    score_refined = Z_test @ w_ref
    return score_proj, score_refined
